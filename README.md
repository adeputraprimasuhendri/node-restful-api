# Node Js Restful API Aplikasi Sekolah
 Restful API with Node Js & MonggoDb

# Requirements
- npm install dotenv
- npm install cors
- npm install body-parser
- npm install mongoose
- npm install express nodemon
- npm install jsonwebtoken --save

# Run App
- npm start
